# AI Chatbot using NLP (Python)

## 📌 Overview
This project is a simple AI chatbot built using Python and Natural Language Processing (NLP) techniques. It processes user input and generates responses using intent-based matching.

## 🚀 Features
- Handles basic conversations
- Uses NLP tokenization (NLTK)
- Lightweight and easy to run
- Extendable to ML-based chatbot

## 🛠️ Tech Stack
- Python
- NLTK (Natural Language Toolkit)

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the chatbot:
   python chatbot.py
